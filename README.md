## 概要
このプログラムは、BMI（Body Mass Index）と体脂肪率に基づいて体型を判別するツールです。体脂肪率と身長、体重を入力として受け取り、その結果をもとに体型（痩せ体型、標準体型、肥満体型）を表示します。また、BMIも計算され、アドバイスが提供されます。
